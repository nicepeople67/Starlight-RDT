Setup guide

Get connected
in minutes.

Step-by-step instructions for every platform. Takes about 2 minutes from download to connected.

1

Download Python first (if you don't have it)

Starlight RDT needs Python to run. If you already have it, skip this step.

↓ Download Python 3.12 for Windows
⚠️

During Python install, check the box that says "Add Python to PATH" on the first screen. This is important.

2

Download Starlight RDT

Click the button below to download the agent. Then extract the zip file anywhere you like.

↓ Download Starlight RDT (zip)
3

Run setup.bat — one double-click

Inside the extracted folder, double-click setup.bat. It automatically installs everything and starts the agent.

⚠️

Windows SmartScreen may warn "unrecognized app". Click More info → Run anyway. This is normal for new software.

✅

A black window opens and shows your session code. Keep this window open while sharing. Close it to stop.

4

Share your session code

The window shows a code like AB3X92DE. Send this to whoever needs to connect. They open Connect Now above and type it in — no download needed on their side.

→ Open Connect page
1

Download Python first (if you don't have it)

python3 --version

If it says "command not found", install Python:

↓ Download Python 3.12 for macOS

Download and extract Starlight RDT

↓ Download Starlight RDT (zip)

Double-click the zip to extract it, then open Terminal and navigate to the folder:

cd ~/Downloads/Starlight-RDT-main

bash setup.sh

This installs dependencies and starts the agent automatically.

⚠️

macOS will ask for Screen Recording and Accessibility permissions. Grant both in System Settings → Privacy & Security. Then re-run bash setup.sh.

4

1

No download needed — just a browser

If someone is sharing their screen with you, you don't need to install anything. Just click Connect Now on this site — works on Chromebook, phone, tablet, any browser.

→ Open Connect page
2

Enter the session code

Ask the person sharing their screen for their session code. Type it into the box on the Connect page and click Connect.

✅

The code is 8 characters, letters and numbers. It looks like AB3X92DE. It's shown in the agent window on the host PC.

3

You're connected — full control

The remote desktop appears. Click to move the mouse, type normally. Use the toolbar for Ctrl+Alt+Del, clipboard, and fullscreen.

💡

On Chromebook: press Ctrl+Shift+F to go fullscreen. Use the Quality dropdown in the toolbar if the connection is slow.

Features

Built right,
not bolted on.

🔒

End-to-end encrypted

All traffic is TLS-encrypted via the relay. Your screen data is never stored or inspected on our servers.

⚡

Hextile encoding

Our custom VNC server uses Hextile tile compression — only changed screen regions are sent each frame.

🌍

Works across networks

Connect from a Chromebook at a coffee shop to a PC at home. Different networks, different ISPs — no problem.

📋

Clipboard sync

Copy on the remote, paste in the browser. Or vice versa. Bidirectional text clipboard sync built in.

🎮

Full keyboard & mouse

Every key — Ctrl, Alt, Win/Cmd, F-keys, special characters — forwarded exactly to the remote machine.

📦

Open source & self-hostable

MIT licensed. Audit the code, run your own relay, or contribute back. No black boxes.
